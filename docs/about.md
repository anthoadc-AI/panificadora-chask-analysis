# About this Project

## The project

The **Panificadora Chask Plant Modernization** was executed by
**INGEDAV S.R.L.** between **21 December 2020** and **4 June 2022** in
Punata, Cochabamba, Bolivia. The intervention covered four phases:

1. **Diagnosis and energy audit** — instrumented monitoring of the
   existing plant to identify inefficiencies.
2. **Design and planning** — CAD-based redesign (SolidWorks) of new
   production areas and equipment.
3. **Implementation** — fabrication and installation of high-efficiency
   machinery (industrial mixer, sheeter, beaters), Energy Management
   System (EMS), and electrical system modernization.
4. **Training and preventive maintenance** — staff training on the new
   equipment and EMS, plus a maintenance program for long-term
   reliability.

## About this repository

This repository was built as a **portfolio-grade reproducible analysis**.
The goals were:

- **Reproduce every claim** of the closure report from the raw dataset.
- **Surface any discrepancies** transparently.
- **Provide multiple access modes**: notebooks (narrative), dashboard
  (interactive), and this docs site (reference + theory).
- **Demonstrate modern Python data engineering practices**: modular
  code, typed signatures, comprehensive tests, CI/CD, auto-generated
  documentation.

## How this site was built

This documentation site is generated by [MkDocs](https://www.mkdocs.org/)
with the [Material](https://squidfunk.github.io/mkdocs-material/) theme.
The API reference pages are auto-generated from Python docstrings using
[mkdocstrings](https://mkdocstrings.github.io/). The site is rebuilt
and redeployed automatically by GitHub Actions on every push to `main`.

To rebuild locally:

```bash
make docs   # serves at http://localhost:8000 with live reload
```
